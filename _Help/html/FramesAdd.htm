Добавление контейнеров

 

Режим предназначен для создания набора "пустых" контейнеров и печати этикеток для оклеивания контейнеров.

Следует ввести количество создаваемых контейнеров. Система создаст заданное число "пустых" (без привязки к товарам и поддонам) контейнеров с последовательными номерами, начиная от максимального существующего.

Сразу же можно напечатать этикетки для созданных контейнеров:

выбрать шаблон этикетки,

указать количество копий для каждой этикетки,

выбрать этикеточный принтер.
