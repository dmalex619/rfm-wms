Контейнеры

 

Товар, размещенный на одном поддоне, предназначенный для размещения в одной ячейке склада, считается контейнером. Каждый контейнер имеет уникальный номер, этот номер (в виде штрих-кода) содержится в этикетке, наклеиваемой на каждый контейнер.

В один контейнер может быть помещен только один товар в состоянии "годен" (монотовар);

недопустимо размещение нескольких разных годных товаров в одном контейнере.

В один контейнер может быть помещено несколько разных товаров в состоянии, отличном от "годен" (например, в состоянии "брак").

 

Набор "пустых" контейнеров создается заранее, для того чтобы напечатать набор этикеток для оклеивания контейнеров. "Пустой" (только что созданный контейнер) не связан ни с каким товаром и поддоном.

"Наполнение" контейнера товаром и определение типа поддона происходит в процессе прихода, когда на паллету с товаром наклеивается этикетка, а пользователь указывает, что конкретному коду на этикетке (т.е. номеру контейнера) соответствует определенный товар в определенном количестве, с указанием наладки (число коробок на паллете, число коробок в ряде) и высоты контейнера.  

 

Контейнеры размещаются в ячейках с признаком "для контейнеров".

Когда контейнер попадает в ячейку, предназначенную для хранения товара без контейнеров, он "расформировывается", т.е. товар, ранее находящийся в контейнере, теперь считается размещенным в ячейке без контейнера, а сам контейнер становится неактуальным (но его номер сохраняется в системе и может быть использован, например, при просмотре истории контейнера).
