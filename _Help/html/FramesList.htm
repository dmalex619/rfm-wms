Список контейнеров

(текущее расположение контейнеров)

 

Экранная форма для отображения списка контейнеров состоит из двух страниц: "Условия" и "Контейнеры".

 

На первой странице "Условия" можно назначить условия, которые ограничат набор выводимых данных:

штрих-код контейнера (вводится с клавиатуры либо сканируется с документа),

владелец контейнера (отметка одного или нескольких значений в списке, либо контекст названия),

состояние контейнера (выбор одного варианта: любое / только созданы / готовы к размещению / перемещаются / на хранении),

срок годности товара в контейнере, до даты;

содержимое контейнера (выбор одного варианта: любое / монотовар (только один товар в контейнере) / разные товары),

товары в контейнере (выбор одного или нескольких значений в специальной форме).

На странице "Контйнеры" будут отображены только те контейнеры, которые удовлетворяют назначенным условиям.

 

Для назначения условий на список контейнеров используются кнопки:

Выбор товаров/упаковок в специальной форме

Очистка всех условий

 

На второй странице "Контейнеры" показаны данные о контейнерах.

Верхняя таблица:

одна строка соответствует одному контейнеру:

уникальный код контейнера (ID),

код контейнера в учетной системе (ERPCode),

владелец контейнера (может быть пуст, т.н. "общий товар"),

состояние товара,

высота контейнера, м,

тип поддона,

состояние контейнера:

      пусто    – только создан, "пустой",

      I           – обрабатывается в приходе,

      T          – участвует в операции транспортировки,

      S          – установлен в ячейку (на хранении),

текущее месторасположение контейнера:

      ячейка (адрес), складская зона и тип зоны.  

Нижняя таблица содержит, в зависимости от выбранной страницы, данные о товарах в контейнере и об истории операций, изменявших состояние контейнера.

Страница "Содержимое":

одна строка соответствует одному товару:

название товара,

количество единиц товара в коробке,

уникальный код контейнера,

владелец,

состояние товара,

количество товара в контейнере:

штук / кг, коробок, паллет,

срок годности (до даты),

товарная группа,

товарный бренд.

Страница "История":

одна строка соответствует одной операции, изменившей состояние / содержимое контейнера (транспортировка, исправление):

дата-время выполнения операции,

тип операции,

пользователь, выполнивший операцию,

устройство, с которого выполнялась операция,

владелец,

состояние товара,

количество товара, участвовавшее в операции

(положительное – товар "добавлен" контейнер, отрицательное – товар "изъят" из контейнера):

штук / кг, коробок, паллет,

срок годности (до даты),

товарная группа,

товарный бренд,

примечание, содержащее информацию вида "откуда, куда, и почему переместился товар".

 

Для работы с контейнерами используются кнопки:

Создать набор новых контейнеров и

сформировать этикетки для печати на этикеточном принтере

Редактировать высоту и тип поддона

Исправить состояние контейнера

Удалить контейнер

 

Сформировать печатные документы:

·         этикетки для печати на этикеточном принтере (по формату)

·         этикетки для печати на обычном принтере (списком)

Дополнительно:

·         Создать операцию транспортировки для текущего контейнера
