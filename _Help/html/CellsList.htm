Список ячеек

 

Экранная форма для отображения списка складских ячеек состоит из двух страниц: "Условия" и "Ячейки".

 

На первой странице "Условия" можно назначить условия, которые ограничат набор выводимых данных:

штрих-код ячейки (полностью или контекст, вводится с клавиатуры либо сканируется),

фиксированный владелец ячейки (отметка одного или нескольких значений в списке),

фиксированное состояние, за которым закреплена ячейка (отметка одного или нескольких значений в списке),

фиксированный товар/упаковка, за которым закреплена ячейка (выбор одного или нескольких значений в специальной форме),

складская зона (отметка одного или нескольких значений в списке),

тип складской зоны (отметка одного или нескольких значений в списке),

блокировка ячейки (выбор одного варианта: не важно / да / нет),

актуальность ячейки (выбор одного варианта: не важно / да / нет),

адрес ячейки (полностью или контекст),

адрес ячейки составной: здание, линия, стояк, уровень, паллетоместо (значение для каждой составляющей выбирается из списка либо вводится с клавиатуры в поле под соответствующим списком),

конкретные ячейки (выбор одного или нескольких значений в специальной форме),

дополнительные условия на страницах:

текущее состояние:

заполнение ячейки (выбор одного варианта: не важно / пустая / заполнена полностью / заполнена не полностью),

товары, находящиеся в ячейке (выбор одного или нескольких значений в специальной форме),

геометрия:

грузоподъемность ячейки, кг (диапазон, от … до …),

высота ячейки, м (диапазон, от … до …),

ширина ячейки, м (диапазон, от … до …),

тип поддона (выбор одного значения),

наличие буферной ячейки (выбор одного варианта: не важно / есть / нет),

история операций в ячейке:

даты операции (диапазон, от … до …).

На странице "Ячейки" будут отображены только те ячейки, которые удовлетворяют назначенным условиям.

 

Для назначения условий на ячейки используются кнопки:

Выбор товаров, ячеек в специальных формах

Очистка всех условий

 

На второй странице "Ячейки" показаны данные о складских ячейках.

Верхняя таблица:

одна строка соответствует одной ячейке:

признак заполненности ячейки (изображение коробки),

признак блокировки ячейки (изображение замочка),

адрес ячейки,

адрес ячейки составной: здание, линия, стояк, уровень, паллетоместо,

складская зона,

тип складской зоны,

фиксированный владелец ячейки,

фиксированное состояние, за которым закреплена ячейка,

фиксированный товар/упаковка, за которым закреплена ячейка,

признак актуальности ячейки,

грузоподъемность, кг,

размеры ячейки, м: высота, ширина,

тип поддона,

адрес буферной ячейки,

максимально возможное число паллет для самой ячейки и для зоны.

уникальный код ячейки (ID),

код ячейки в учетной системе (ERPCode).

Нижняя таблица содержит, в зависимости от выбранной страницы, данные о содержимом ячейки (т.е. о товарах / контейнерах, находящихся в ячейке) и об истории операций, изменявших состояние ячейки.

Страница "Содержимое":

одна строка соответствует одному товару / контейнеру:

название товара и вложение (количество единиц товара в коробке),

количество товара в контейнере / ячейке:

штук / кг, коробок, паллет,

срок годности (до даты),

товарная группа,

товарный бренд.

Страница "История":

одна строка соответствует одной операции, изменившей состояние / содержимое ячейки (транспортировка, исправление):

дата-время выполнения операции,

тип операции,

пользователь, выполнивший операцию,

устройство, с которого выполнялась операция,

уникальный код контейнера,

владелец,

состояние товара,

количество товара, участвовавшее в операции

(положительное – товар поступил в ячейку, отрицательное – товар убыл из ячейки):

штук / кг, коробок, паллет,

срок годности (до даты),

товарная группа,

товарный бренд,

примечание, содержащее информацию вида "откуда, куда, и почему переместился товар".

 

Для работы с ячейками используются кнопки:

Добавить одну ячейку

Добавить диапазон ячеек (с автоматическим формированием адреса)

Редактировать адрес ячейки

Редактировать геометрические параметры и грузоподъемность ячейки

Редактировать фиксированные закрепления ячейки:

владелец, состояние товара, товар/упаковка

Исправить состояние ячейки

Удалить ячейку

 

Сформировать печатные документы:

·         этикетки для печати на этикеточном принтере (по формату)

·         этикетки для печати на обычном принтере (списком)

Дополнительно:

·         Блокировать ячейку / Снять блокировку ячейки

·         Установить актуальность ячейки / Снять актуальность ячейки

·         Назначить буферную ячейку

 

 

Добавление одной ячейки

 

Используется для добавления единичных ячеек.

Для добавления ячейки следует указать данные:

складская зона,

составляющие адреса: 

здание (1 символ),

линия (1 латинская заглавная буква),

стояк (2 цифры, с ведущим нулем),

уровень (1 цифра),

паллетоместо (1 цифра).

адрес (может быть сформирован из составляющих при нажатии на кнопку с изображением зеленых стрелочек либо введен с клавиатуры при включенном флажке "изменить адрес").

Составляющие адреса могут быть выбраны из ниспадающих списков, состоящих из существующих значений, либо введены в текстовые поля под соответствующими списками (в текстовые поля могут быть введены как существующие, так и новые значения).

При добавлении единичных ячеек какие-либо составляющие адреса могут быть пусты.

 

При сохранении проверяется, чтобы введенный адрес был уникальным.

 

При добавлении ячейки используются кнопки:

Построить адрес из составляющих

Сохранить (создать новую ячейку)
