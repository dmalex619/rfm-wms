Стандартная форма для работы с таблицами простой структуры

 

Для работы с таблицами простой структуры (например, с таблицами-классификаторами) используется стандартная форма, содержащая элементы:

·          список таблиц, доступных для отображения в данном режиме,

·          таблица с данными,

·          кнопки для работы с данными.

 

В списке, расположенном в верхней части формы, приведены русские и латинские названия таблиц данных,  разрешенных для отображения в данном режиме.

Следует выбрать таблицу в списке и нажать расположенную справа от списка кнопку <Обновить>. 

Таблица заполняется данными, строки расположены в порядке ввода (т.е. в порядке возрастания внутреннего уникального ключа ID).

 

Кнопки для работы в стандартной форме отображения таблицы:

Добавление новой записи в таблицу

Добавление новой записи путем копирования текущей записи

Редактирование текущей записи

Удаление текущей записи

 

Если выбранная таблица данных разрешена для отображения, но не размещена для изменения, кнопки будут недоступны.

 

Для ввода и редактирования используется стандартная форма отображения записи в паспортном виде.
