Условие: отметка нескольких значений

 

Для отметки нескольких значений из связанной таблицы-классификатора используется специальная форма, которая вызывается нажатием на кнопку с изображением лупы.

В появившейся форме следует найти (для поиска можно использовать стандартные режимы поиска в таблице и местные фильтры) и отметить все нужные элементы, а затем нажать кнопку с изображением желтой молнии.

Список отмеченных значений будет передан в исходную форму.
