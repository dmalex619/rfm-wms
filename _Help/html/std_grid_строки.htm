Строки, колонки, ячейки таблицы

 

Одна строка таблицы соответствует одной записи данных.

 

Самая верхняя служебная строка таблицы, выводимая на сером фоне, содержит заголовки (названия) колонок.

Колонки с краткими названиями могут быть снабжены примечаниями, которые можно увидеть, если подвести указатель мыши к заголовку колонки.

 

Таблица может иметь нижнюю итоговую строку, содержащую суммы для числовых полей;

итоговая строка показывается жирным шрифтом на сером фоне. 

 

Самая левая служебная колонка таблицы, выводимая на сером фоне, содержит указатель на текущую строку.

 

Таблица может иметь левую колонку пометки, содержащую флажки для отметки записей (для массовых операций); установленный флажок означает, что запись отмечена (выбрана).
