Итоговая строка (строка статуса)

 

Для некоторых таблиц предусмотрена возможность отображения итоговой строки, содержащей суммы по числовым колонкам.

Итоговая строка всегда является последней (нижней) и отображается жирным шрифтом на сером фоне.

 

Для отображения итоговой строки таблицы следует щелкнуть правой кнопкой мыши на территории таблицы и в появившемся меню выбрать пункт "Показать итоговую строку".

 

Для отображения таблицы без итоговой строки следует щелкнуть правой кнопкой мыши на территории таблицы и в появившемся меню выбрать пункт "Скрыть итоговую строку".
