Экспорт в Excel

 

Пользователь может вывести в MS Excel любую отображаемую в экранной форме таблицу.

Каждая строка таблицы на экране будет представлена в виде строки Excel,

каждая выбранная для экспорта колонка – в виде колонки Excel.

Для передачи данных в Excel следует щелкнуть правой кнопкой мыши на территории таблицы и в появившемся меню выбрать пункт "Экспорт в Excel".

Записи будут выведены в том порядке (с той же сортировкой), как они показаны в таблице.

 

В левом списке показаны все колонки текущей таблицы, данные которых могут быть переданы в Excel.

 

С помощью кнопок со стрелками влево-вправо, находящихся в центральной части формы, следует перенести в правую часть формы те названия колонок, которые следует передать в Excel:

кнопка с двумя стрелками вправо          - включить в экспорт все колонки,  

кнопка с одной стрелкой вправо                        - включить в экспорт текущую колонку,

кнопка с двумя стрелками влево                        - исключить из экспорта все выбранные колонки,  

кнопка с одной стрелкой влево                         - исключить из экспорта текущую выбранную колонку.

 

С помощью кнопок со стрелками вверх-вниз, находящихся под правым списком, следует расположить выбранные  колонки в том порядке, как должны быть расположены колонки на листе Excel:

кнопка со стрелкой вверх          – передвинуть текущую колонку на предыдущую позицию,

кнопка со стрелкой вниз            – передвинуть текущую колонку на следующую позицию.  

 

Если исходная таблица содержит колонку пометки и есть отмеченные записи, в Excel могут быть экспортированы:

·         только записи с отметками, если флажок "только помеченные записи" установлен,

·         все записи, независимо от отметок, если флажок "только помеченные записи" снят.

 

По нажатию кнопки с изображением значка Excel система создает новый Excel-документ с данными.

Первая строка документа Excel содержит данные о текущей дате-времени, имя пользователя и название текущего режима; следующая строка содержит заголовки колонок, выделенные жирным шрифтом.

Если исходная таблица содержит итоговую строку, эта строка также будет выведена в Excel.

Размер, цвет и формат шрифта, используемые в исходной таблице на экране, в Excel не передаются.
