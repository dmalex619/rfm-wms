Список предопределенных значений

(с выбором одного значения)

 

Поле, позволяющее выбрать одно из предопределенных значений, входящих в список.

Если пользователь нажмет местную стрелочку (маленькую кнопочку справа от поля, аналогично нажатию клавиши <F4>), он получит список всех допустимых значений, упорядоченный по алфавиту.

Для перемещения в этом списке пользователь может использовать клавиши управления курсором,

перемещать бегунок в правой части списка,

набирать на клавиатуре первый символ искомого значения (последовательное нажатие одного и того же символа перемещает текущую строку по элементам, начинающимся с этого символа).

 

Меню на правой кнопке мыши для списка

 

Если щелкнуть правой кнопкой мыши на территории закрытого списка, появится меню, содержащее пункты для вызова дополнительных действий:

·          в начало (выбор первого значения в списке) / в конец (выбор последнего значения в списке),

·          обновление данных списка,

·          поиск,

·          очистка (выбор пустого значения).

В качестве первого пункта меню на правой кнопке всегда отображается количество элементов в списке.

 

Поиск значения в списке                                              

 

Найти нужную запись в списке можно с помощью режима "Поиск", для вызова которого следует щелкнуть правой кнопкой мыши на территории закрытого списка и в появившемся меню выбрать пункт "Поиск" (аналогично нажатию клавиши <F7>).

 

В форме "Поиск ..." пользователь указывает значение для поиска и может указать дополнительные параметры поиска (отображаются при нажатии на кнопку <Параметры>).

 

Если флажок þ "От указателя" установлен, значит, система будет искать подходящий элемент списка, начиная с текущего;

если нет, значит, система будет выполнять поиск по всем элементам списка, начиная с первого (по умолчанию).

Если флажок þ "Регистр" установлен, значит, поиск ведется с учетом регистра (большие / маленькие буквы); если нет, значит, регистр не имеет значения (по умолчанию).

Если флажок þ "Контекст" установлен, значит, система будет искать такой элемент списка, значение которого содержит поисковое значение в любом месте, как контекст (по умолчанию), таким образом по контексту "мол" будут найдены значения "Молочный", "Смоленск" и. т.д.;

если нет, значит, система будет искать такой элемент списка, где значение полностью совпадает с поисковым значением.

 

При нажатии на кнопку с изображением бинокля система начинает искать элемент списка, удовлетворяющий введенному условию, и если находит, то выбирает его в качестве текущего значения списка.

Если подходящего элемента нет, система выдает сообщение "Значение не найдено!".

 

Продолжение поиска

После того, как система нашла элемент списка, удовлетворяющий условию поиска, можно продолжить поиск по тому же условию. Для продолжения поиска следует сновать вызвать меню на правой кнопке мыши и выбрать пункт "Продолжить поиск" (аналогично нажатию клавиши <F8>).
