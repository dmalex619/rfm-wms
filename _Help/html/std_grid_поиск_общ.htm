Поиск по произвольной колонке

 

Найти нужную запись в таблице можно с помощью режима "Поиск", для вызова которого следует щелкнуть правой кнопкой мыши на территории таблицы и в появившемся меню выбрать пункт "Поиск" (аналогично нажатию клавиш <Ctrl>+<F> или <F7>).

Условие для поиска можно наложить только на одну колонку таблицы (на одно поле).

 

В форме "Поиск ..." пользователь выбирает колонку, для которой задается условие (по умолчанию – текущая колонка), и указывает значение для поиска, а также может указать дополнительные параметры поиска (отображаются при нажатии на кнопку <Параметры>).

 

Если флажок þ "От указателя" установлен, значит, система будет искать запись, начиная с текущей строки;

если нет, значит, система будет искать запись по всей таблице, начиная с первой строки (по умолчанию).

Если флажок þ "Регистр" установлен, значит, поиск ведется с учетом регистра (большие / маленькие буквы); если нет, значит, регистр не имеет значения (по умолчанию).

Если флажок þ "Контекст" установлен, значит, система будет искать такую запись, где выбранное поле содержит поисковое значение в любом месте, как контекст (по умолчанию), таким образом по контексту "мол" будут найдены значения "Молочный", "Смоленск" и. т.д.;

если нет, значит, система будет искать такую запись, где выбранное поле полностью совпадает с поисковым значением.

 

При нажатии на кнопку с изображением бинокля система начинает искать запись, удовлетворяющую введенному условию, и если находит, то устанавливает на нее активную строку.

Если подходящей записи нет, система выдает сообщение "Значение не найдено!".

 

Продолжение поиска по произвольной колонке

После того, как система нашла запись, соответствующую условию поиска, можно продолжить поиск по тому же условию. Для продолжения поиска следует сновать вызвать меню на правой кнопке мыши и выбрать пункт "Продолжить поиск" (аналогично нажатию клавиш <Ctrl>+<G> или <F8>).
