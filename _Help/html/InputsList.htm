Список приходов

 

Экранная форма для отображения списка приходов состоит из двух страниц: "Условия" и "Приходы".

 

На первой странице "Условия" можно назначить условия, которые ограничат набор выводимых данных:

штрих-код прихода (вводится с клавиатуры либо сканируется с документа),

дата прихода: диапазон, с … до …,

тип прихода (выбор одного значения: основной, возврат, …),

поставщик (отметка одного или нескольких значений в списке, либо контекст названия),

владелец товара (отметка одного или нескольких значений в списке, либо контекст названия),

состояние прихода (выбор одного варианта: все / подтверждены / не подтверждены),

состояние контейнеров в приходе (выбор одного варианта: все / проблемы / подготовлены, не размещены / размещены),

товары, участвующие в приходе (выбор одного или нескольких значений в специальной форме).

На странице "Приходы" будут отображены только те приходы, которые удовлетворяют назначенным условиям.

 

Для назначения условий на приходы используются кнопки:

Выбор товаров/упаковок в специальной форме

Очистка всех условий

 

На второй странице "Приходы" показаны данные о приходах.

Верхняя таблица:

одна строка соответствует одному приходу:

уникальный код прихода (ID),

код прихода в учетной системе (ERPCode),

дата прихода,

тип прихода,

поставщик,

владелец,

состояние товара (ожидаемое),

примечание,

для обрабатываемых или обработанных приходов:

время начала обработки прихода,

пользователь,

мобильное устройство,

для подтвержденных приходов:

отметка подтверждения (черная галочка),

время подтверждения прихода.

Нижняя таблица содержит, в зависимости от выбранной страницы, данные о товарах, контейнерах и таре в приходах.

Страница "Товары":

одна строка соответствует одному товару в конкретном состоянии:

название товара,

количество единиц товара в коробке,

количество коробок на паллете,

состояние товара,

количество заказанного товара ("Заказ …"):

штук / кг, коробок, паллет,

количество фактически поступившего товара ("Пришло …"):

штук / кг, коробок, паллет,

отметка соответствия заказанного и фактически поступившего товара, с учетом состояния товара:

зеленый шарик:           товара поступило ровно столько, сколько заказано, 

желтый шарик:            товара поступило больше, чем заказано (возможен пересорт),

красный шарик:            товара поступило меньше, чем заказано.

Страница "Товары (итого)":

одна строка соответствует одному товару, без учета состояния:

название товара,

количество единиц товара в коробке,

количество коробок на паллете,

количество заказанного товара ("Заказ …"):

штук / кг, коробок, паллет,

количество фактически поступившего товара ("Пришло …"), сумма по всем состояниям:

штук / кг, коробок, паллет.

Страница "Контейнеры":

одна строка соответствует одному контейнеру:

уникальный номер контейнера,

название товара,

количество единиц товара в коробке,

количество коробок на паллете,

состояние товара,

фактическое количество товара в контейнере:

штук / кг, коробок, паллет,

срок годности (до даты),

высота контейнера (м),

текущая ячейка, где в настоящий момент расположен контейнер:

адрес, складская зона, тип складской зоны,

отметка подбора ячейки для хранения контейнера и подготовки операций транспортировки контейнера в подобранную ячейку (черная галочка),

отметка выполнения подготовленных операций транспортировки контейнера в зону хранения:

зеленый значок:          операция транспортировки выполнена,

контейнер находится за пределами приемной зоны,

красный значок:           операция транспортировки не выполнена.

Страница "Тара":

одна строка соответствует одному наименованию тары:

тип тары,

количество фактически поступивших единиц тары.


 

Для работы с приходами используются кнопки:

Создать новый приход

Редактировать приход

(только для неподтвержденных приходов)

Подтвердить выполнение прихода

Удалить приход

(только для неподтвержденных приходов)

 

Сформировать печатные документы:

·         Приходная накладная, со штрих-кодами (заказанное количество товара)

·         Приходная накладная, со штрих-кодами (заказанное количество товара, по паллетам)

·         Приходная накладная, с учетом состояния товара (заказанное и фактически поступившее количество товара, для каждого состояния)

·         Приходная накладная, без учета состояния товара (заказанное и фактически поступившее количество товара,  сумма по всем состояниям)

·         Акт сверки (заказанное и фактически поступившее количество товара, для каждого состояния) – только для товаров, по которым имеются несоответствия

Дополнительно:

·         Выполнить повторное размещение контейнеров (для приходов, в которых есть проблемы с размещением контейнеров: не подобраны ячейки),

·         "Разблокировать" приход (снять отметку о начале обработки прихода)
