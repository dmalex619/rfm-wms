Редактирование фиксированных закреплений ячейки

 

Изменение фиксированных закреплений возможно только для пустых ячеек.

 

Изменение фиксированных закреплений ячейки за владельцем, состоянием товара и товаром/упаковкой выполняется либо для текущей ячейки (если в таблице нет отмеченных ячеек), либо для всех отмеченных ячеек.

При изменении закреплений для одной ячейки следует выбрать значения для каждого нужного типа закрепления;

при изменении закреплений для нескольких отмеченных ячеек для каждого типа закрепления следует включить соответствующий флажок, разрешающий массовую замену данных, и выбрать общее для всех изменяемых ячеек значение.

 

Для ячейки задаются фиксированные закрепления:

владелец (выбор одного значения; очистка назначения по кнопке с красным крестом),

состояние товара (выбор одного значения; очистка назначения по кнопке с красным крестом),

товар/упаковка (выбор одного значения; очистка назначения по кнопке с красным крестом).

Закрепление ячейки за владельцем означает, что в этой ячейке не может быть размещен контейнер/товар другого владельца или без указания владельца.

Закрепление ячейки за состоянием товара означает, что в этой ячейке не может быть размещен контейнер/товар в другом состоянии.

Закрепление ячейки за товаром/упаковкой означает, что в этой ячейке не может быть размещен другой товар/упаковка.

Несколько ячеек хранения могут быть закреплены за одним владельцем, состоянием, товаром/упаковкой;

только одна ячейка пикинга может быть закреплена за одним владельцем, состоянием, товаром/упаковкой. 

 

При редактировании фиксированных закреплений ячейки используются кнопки:

Выбрать товар/упаковку (в специальной форме)

Очистить:

закрепление за владельцем,

закрепление за состоянием товара,

закрепление за товаром / упаковкой

Сохранить закрепления
