Меню на правой кнопке мыши

 

Если щелкнуть правой кнопкой мыши на любой ячейке таблицы, появится меню, содержащее пункты для вызова дополнительных действий в таблице:

·          перемещение в начало таблицы (на первую строку) / в конец таблицы (на последнюю строку),

·          обновление данных таблицы,

·          отображение только актуальных записей / всех записей,

·          фиксация положения (заморозка) колонок, 

·          поиск и продолжение поиска,

·          экспорт данных в Excel,

·          отображение итоговой строки,

·          отображение колонки пометки,

·          …

Набор пунктов в меню на правой кнопке может быть различен для разных таблиц и определяется их прикладным назначением.

 

В качестве первого пункта меню на правой кнопке всегда отображается количество записей в таблице (не считая итоговой строки).

Если в таблице используется колонка пометки, в скобках показано количество отмеченных записей.
