Редактирование геометрических параметров контейнера: высоты и типа поддона

 

Изменение высоты и типа поддона выполняется либо для текущего контейнера (если в таблице нет отмеченных контейнеров), либо для всех отмеченных контейнеров.

При изменении параметров для одного контейнера следует ввести значения для каждого параметра;

при изменении параметров для нескольких отмеченных контейнеров для каждого изменяемого параметра следует включить соответствующий флажок, разрешающий массовую замену данных, и ввести общее для всех изменяемых контейнеров значение параметра.

 

Для контейнера задаются параметры:

тип поддона (выбор одного значения),

высота контейнера, м.
